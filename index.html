Como usar o Script Tibia OTC

Guia de Utilização

Após instalar o script usando o programa, siga estas instruções para utilizá-lo corretamente:

  1. Inicie o Tibia OTC e faça login em seu personagem.
  2. Irá carregar um button escrito "MACROS". Nele você gerencia tudo!
  3. Configure as opções de acordo com sua necessidade.
  4. Para ADICIONAR um macro, basta clicar nele para ficar verde e clique em "Adicionar Macros".
  5. Para ADICIONAR mais de 1 macro, basta selecionar varios macros nas abas e clicar em "Adicionar Macros".
  6. Para REMOVER macros, basta clicar nele para ficar vermelho e click em "Adicionar Macros". Assim, só adicionará os macros que estão verdes.
  7. RECOMENDADO: Utilize o BOT COMPLETO, ele já vem todo configurado. Se não quiser e apenas quiser adicionar os macros conforme sua escolha, pressione "Desligar Tudo" e coloque seus macros de preferencia.
  8. NECESSÁRIO: Se for alterar o Bot Completo Main para o Bot Completo GW, primeiro click em "Desligar Tudo" e apenas depois coloque o outro bot completo. Tanto para o Main quando para o GW.

Recursos Disponíveis

  • FULL UPP 100% AFK: SISTEMA DE UP AVANÇADO QUE FAZ TUDO AUTOMATICO
  • SCRIPT BOT GW: FOCADO EM GUERRAS
  • ICONES PERSONALIZADOS: ICONES PARA QUASE TODOS OS MACROS ⚠️ AINDA EM DESENVOLVIMENTO ⚠️

Como instalar o Script

Baixar Script Agora

Instalação Passo a Passo

  1. Faça o download do arquivo de instalação
  2. Para instalar basta clicar em avançar > avançar > instalar > concluir
  3. Espere até que o programa abra
  4. Após o programa abrir terá 5 botões na parte centro/baixo do pragama.
  5. Botão "Criar Bot" já cria automaticamente o seu bot. Se quiser, já pode usar assim. Basta ir no seu jogo, reiniciar o bot dando 0FF/ON e colocar o bot que você criou!
  6. Se for utilizar para upar, é necessária aperta o botão "Gerenciar caves". Lá você tem que marcar primeiramente a opção "ESSENCIAIS". Em seguida, adiciona onde você quer upar. Se quiser adicionar tudo, pode adicionar. Se quiser apenas adicionar as caves do seu reset, tudo bem também. O recomendo é adicionar tudo!
  7. Após criar a pasta, adicionar as caves, você já está pronto para usar o bot. Basta da OFF/ON no bot e colocar a pasta que você criou.

Requisitos do Sistema

  • Client do Spartacus OT ATUALIZADO!
  • Sistema Operacional: Windows 7 ou superior
  • Já ter acessado o Spartacus OT antes de usar o pragrama
  • Antivirus: Pode ser necessário fazer a liberação para o programa
