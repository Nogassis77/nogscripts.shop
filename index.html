Como Usar o Script Tibia OTC

Guia de Utilização

Após instalar o script, siga estas instruções para utilizá-lo corretamente:

  1. Inicie o Tibia OTC e faça login em seu personagem
  2. Pressione a tecla F2 para abrir o menu de configurações
  3. Configure as opções de acordo com sua necessidade
  4. Salve as configurações e feche o menu
  5. Use F3 para ativar/desativar recursos rapidamente
  6. Use F4 para abrir o menu de cavebot

Recursos Disponíveis

  • Cave Bot Avançado: Sistema automático de caça
  • Target System: Priorização de alvos
  • HUD Personalizável: Interface adaptável
  • Auto Healer: Cura automática
  • Auto Rune Maker: Fabricação de runas
  • Loot Seller: Venda automática de loot

Como Instalar o Script

Instalação Passo a Passo

  1. Faça o download do arquivo de instalação
  2. Feche completamente o cliente do Tibia
  3. Extraia o conteúdo do arquivo ZIP em uma pasta dedicada
  4. Execute o arquivo "Instalador.exe" como administrador
  5. Siga as instruções do assistente de instalação
  6. Inicie o Tibia OTC após a conclusão

Requisitos do Sistema

  • Tibia OTC instalado e atualizado
  • Sistema Operacional: Windows 7 ou superior
  • Memória RAM: 4GB mínimo (8GB recomendado)
  • Antivirus: Pode ser necessário adicionar exceção
